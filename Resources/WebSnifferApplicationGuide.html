Web Sniffer Application Guide

About application

This application lets you search across different websites within an unified interface. This can be immensely useful for research purposes. This tool was created keeping teachers and learners in mind.

The app also allows saving pages for offline use.

Application features

  • Download pages for offline use.
  • Uses chromium browser instance for speedy navigation.
  • Tabbed interface and asynchronous loading.
  • Light on memory and excludes unnecessary features.
  • Completely customizable.
